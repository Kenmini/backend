# Presentation diagrams

This directory contains the reproducible Mermaid sources for four presentation-focused diagrams in English and Japanese. Each diagram can be rendered in light and dark themes.

```powershell
.\scripts\render-presentation-charts.ps1
```

The command creates SVG and PNG files in `images/charts/`. That output directory is intentionally ignored because the generated assets are local presentation artifacts; the Mermaid sources, theme files, and pinned renderer are committed.
